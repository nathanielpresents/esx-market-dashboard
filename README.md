## Overview

A professional-grade interactive dashboard built to support equity research, valuation modeling, and market analysis for the Ethiopian Securities Exchange (ESX).

This tool brings together institutional-grade financial modeling with user-friendly interfaces, designed for analysts, investors, and financial professionals focusing on frontier markets.

## Features

- **Market Overview** — Key metrics, sector performance, and market snapshots
- **Equity Valuation Tool** — Interactive DCF / DDM models with real-time sensitivity analysis
- **Credit Risk Engine** — Integration with EthioScore alternative credit scoring framework (coming soon)
- **Portfolio Simulator** — Track and simulate investment portfolios on ESX (coming soon)
- **Professional-grade assumptions** grounded in real Ethiopian company financials

## Valuation Models

- Dividend Discount Model (DDM)
- Sensitivity analysis on growth rates and required returns
- Clear BUY / undervaluation signals
- Data sourced from company reports and ESX listings

## Tech Stack

- **Framework**: Streamlit
- **Visualization**: Plotly
- **Data**: Pandas
- **Deployment**: Streamlit Community Cloud
